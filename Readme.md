## Overview

This project is a Machine Learning-based Car Price Prediction System developed using Python, Scikit-learn, and Streamlit. The application predicts the estimated selling price of a used car based on factors such as vehicle age, present price, kilometers driven, fuel type, seller type, transmission type, and ownership history.

## Features

* Interactive Streamlit Dashboard
* Real-Time Car Price Estimation
* Data Preprocessing and Feature Engineering
* Machine Learning Regression Model
* User-Friendly Interface
* Instant Price Prediction

## Technologies Used

* Python
* NumPy
* Pandas
* Scikit-learn
* Streamlit
* Matplotlib

## Dataset

The model is trained on a used car dataset containing information such as:

* Year of Purchase
* Present Price
* Kilometers Driven
* Fuel Type
* Seller Type
* Transmission Type
* Number of Owners

## Project Workflow

1. Data Collection and Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Model Training and Evaluation
5. Dashboard Development using Streamlit
6. Real-Time Price Prediction

## Installation

Clone the repository:

```bash
git clone https://github.com/Saurabh-Singh68/Car-Price-Prediction-Dashboard.git
```

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
